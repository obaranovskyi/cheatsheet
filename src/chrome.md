# Tabs

- `Cmnd+n` - Open window number `n`
- `Cmnd+Shit+t` - Open previously closed tab
- `Cmnd+t` - Open new tab
- `Cmnd+w` - Close tab

# History

- `Cmnd+[` - Go back by history
- `Cmnd+]` - Go forward by history
